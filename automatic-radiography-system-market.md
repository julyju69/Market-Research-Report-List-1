<p><p>Automatic Radiography System Market Types include X-ray Radiographic and Gamma-ray Radiographic. X-ray Radiographic is a type of automatic radiography system that uses X-rays to create detailed images of the internal structures of an object or a person's body. X-rays pass through the object and are captured on film or electronically to produce the image. On the other hand, Gamma-ray Radiographic is a type of automatic radiography system that uses gamma rays, a form of electromagnetic radiation, to generate images. It is particularly useful for inspecting thick and dense materials such as steel or concrete.</p></p>
